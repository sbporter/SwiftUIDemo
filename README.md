# SwiftUIDemo

This is the completed demo from the SwiftUI Overview from GDG Salt Lake - Mobile & ML Devfest in September 2019

## Requirements

- Xcode 11.0+
- Swift 5.1+
- Live Preview requires MacOS 10.15 Catalina
